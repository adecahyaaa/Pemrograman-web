Keahlian

Pada praktikum ini, saya mempelajari bagaimana pengaplikasian tag semantic pada html. Semantic yang saya aplikasikan antara lain:

  • tag p: untuk paragraf
  • tag h1, h2, h3: untuk header. h1 artinya header1, dan seterusnya. Semakin meningkat angka header, maka tulisan akan semakin kecil
  • tag img: untuk menampilkan gambar. menggunakan beberapa atribut, yaitu:
    • src (source): sumber dari gambar yang akan ditampilkan.
    • atribut alt (alternatif):alternatif text yang akan ditampilkan jika gambar dari src tidak berhasil dimuat.
  • Form: untuk membuat form yang atraktif. dapat menerima input berupa email dan atau pesan. Tag yang digunakan:
    • tag form
    • tag label: untuk melabeli kolom inputan
    • input: untuk mengambil input dari user. Dapat ditambah atribut 'type' untuk mengetahui bentuk input yang harus dimasukkan (seperti type email, type submit)
    • textarea: untuk mengambil input dengan bentuk form text yang dimasukkan banyak
